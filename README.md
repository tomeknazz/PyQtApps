# PyQtApps

This repository contains a collection of desktop applications developed using Python and the PyQt framework. 

## Academic Context

These projects were created as part of the coursework for the Architecture of Computer Systems (Architektura Systemów Komputerowych) class at Gdańsk University of Technology (Politechnika Gdańska). 

Please note that while this README is in English, the graphical user interfaces (GUI) of the applications, as well as the internal logic and documentation for these specific assignments, are implemented in the Polish language.

## Technologies Used

* **Python 3.x**: The core programming language used for logic and backend operations.
* **PyQt6**: The primary framework utilized for designing and rendering the graphical user interfaces.

## Getting Started

### Prerequisites

To run these applications, you must have Python installed on your local machine.

### Installation

1. Clone the repository to your local environment:
   ```bash
   git clone [https://github.com/tomeknazz/PyQtApps.git](https://github.com/tomeknazz/PyQtApps.git)
   ```

2. Navigate into the cloned directory:
   ```bash
   cd PyQtApps
   ```

3. Install the required graphical libraries. It is highly recommended to use a virtual environment:
   ```bash
   pip install PyQt6
   ```

## Usage

Each application is contained within its respective directory. To launch a specific application, navigate to its folder and execute the main Python script. 

For example:
```bash
python main.py
```
